Luciel's HTML & CSS Cheat Sheet

HTML Elements

HTML elements make up what's on a webpage. They're comprised of an opening tab, the content between the tags, and a closing tag, which is the same as the opening tag, but with a forward slash in front of it. HTML Elements can also have attributes added onto them which can provide information or change styling. These attributes are id, which is represented by "#"; class, which is represented by "."; src, which sets an image or video's source; alt, which is exclusive to images and provides alternate text describing it for people with screen readers; href, which links an HTML element to another page or a different part of the page they're on; target, which tells the computer where the link should open; controls, for video only which displays a pause and play button automatically, along with a volume control and fullscreen option; autoplay, only for video, which automatically plays the video once the webpage loads; loop, only for video, which allows the video to continuously play on repeat; and attributes that alter the style of the text within the HTML file, such as width, height, etc.

Tag Name Description

< head > Head Contains metadata for the page that isn't displayed on the page itself.
< title > Title The title that will be displayed in the tab of the webpage when you open it.
< body > Body Contains all of the visible content of the webpage.
< h1 > - < h6 > Heading 1 - Heading 6 Goes from h1-h6; Used to title pages or sections of pages.
< p > Paragraph Contains a plain block of text, used for long stretches of text.
< div > Division Divides the page into sections.
< span > Span Contains short pieces of text or other HTML.
< em > Emphasis Emphasizes a piece of text by putting it in italics.
< strong > Strong Emphasizes a piece of text by putting it in bold text.
< br > Line Break Creates a line break inside of an element.
< ul > Unordered List A bulletted list of items in no particular order.
< ol > Ordered List A numbered list of items in a particular order.
< li > List Item Items listed in an ordered or unordered list.
< img > Image Allows you to add an image to the page.
< video > Video Allows you to add a video to the page.
< a > Anchor Allows you to link other websites or parts of the page itself inside of other elements.
< link > Link Adds a plain link to the page.

HTML Tables

HTML tables create a visual way of displaying data in a tabular form. HTML tables also have attributes, like scope, which makes it clear to the program that a heading is for either a column or a row; border, which adds a border between the lines of the table, making it easier to read; and colspan and rowspan, which allow data to span across more than one column or row.

Tag Name Description
< table > Table A container for tabular data.
< thead > Table Head Contains the table's headings.
< tbody > Table Body Contains the main contents of the table.
< tr > Table Row Creates a new row in a table.
< td > Table Data Adds data within the cells of the table.
< th > Table Heading Describes what data you're looking at.
< tfoot > Table Footer At the very bottom of the table; usually contains sums, differences, and other data results.

Semantic HTML

Semantic HTML is HTML that has meaning. The tags represent what part of the text the HTML is making up. Only a few have specific attributes. These attributes are for audio and they are type, which tells the computer what type of audio file it is and controls, which automatically display the audio controls into the browser, such as play and mute.

Tag Name Description
< header > Header Container for navigation links or an introductory message.
< nav > Navigation Used to define menus or a table of contents.
< main > Main Establishes the main content of a webpage.
< footer > Footer Contains contact information, copyright information, terms of use, a site map, and/or a reference to top of the page links.
< article > Article Defines parts of the document with the same theme.
< section > Section Holds content that can make sense without extra context.
< aside > Aside Marks a supplemental piece of information that isn't necessary to understand the topic at hand.
< figure > Figure Used to encapsulate most forms of media like images, diagrams, etc.
< figcaption > Figure Caption Adds a caption to the figure, usually inside of the figure itself.
< embed > Embed Able to embed any media content into the page.

CSS Selectors & Specificity

CSS selectors are the part of CSS code that targets a piece of HTML code. It's the first thing you type when getting ready to make a CSS declaration. There are several different types of selectors that CSS uses, and each one has a different specificity in the code. The specificity of a selector tells the browser which CSS style to apply first.

  • Universal: Universal selectors target every HTML element in the HTML document. This is the least specific selector and is very easy to override with other selectors.
  • Type: Type selectors exclusively use HTML elements, but without the brackets around them. It's also sometimes called the "tag name selector" or "element selector". This is one of the least specific selector and will be overshadowed by almost any other selector.
  • Attribute: The attribute selector selects attributes within elements and changes their appearance on the page.
  • Class: Class selectors select elements of a certain class, which is contained within the tag as an attribute. You can apply multiple classes to a singular element by putting a space inbetween them. Classes can be applied to many different elements and are slightly more specific than the Type selector.
  • ID: ID selectors are almost the most specific selector you can use in CSS. It selects all elements of a certain ID. ID's are only able to be applied to one element, making it very specific.
  • !important: Adding this into your CSS declaration will prioritize that stylization over any other stylization. Use sparingly.
  • Pseudo-Class: Pseudo-Class selectors are applied after one of the other selectors to change the appearance of an element after an interaction. These include, :hover, :focus, :visited, :disabled, and :active.
  • Chaining: Chaining allows you to give an element more than one CSS selector at a time.
  • Descendant Combinator: This allows you to only select elements within a different element, or the "children" of that element.
  • Multiple Selectors: This allows you to choose multiple selectors at a time to change what they look like. You do this by simply adding a comma between the selectors.

The Box Model

The Box Model is comprised of set properties that tell you how much space an element takes up on a page.

Property Name Description
height Height Specifies the height of an element on a page
width Width Specifies the width of an element on a page.
border Border A line that surrounds an element; Able to change the width, style, and color.
border-radius Border Radius Modifies the roundness of the corners of an element's border.
padding Padding The spacing between the content of an element and the borders of the element.
margin Margin The spacing between the borders of an element and the edge of the page.
min-width Minimum Width The minimum width you want an element to be displayed with when the page is made smaller.
min-height Minimum Height The minimum height you want an element to be displayed with when the page is made smaller.
max-width Maximum Width The maximum width you want an element to be displayed with when the page is made bigger.
max-height Maximum Height The maximum height you want an element to be displayed with when the page is made bigger.
overflow Overflow Controls what happens when content spills out of the parent element. Either hidden, scroll, or visible.
visibility Visibility Decides whether an element is visible or not. Either hidden, visible, or collapse.

CSS Display & Positioning

Display and Positioning in CSS, like you'd expect, changes the display or position of HTML elements. Positions can be specified in 5 different ways. Static, the default position; Relative, where you reposition the element relative to its spot on the page; Absolute, which will basically cause all other elements on the page to act like it isn't there; Fixed, which fixes an element to a spot on the page so that it still appears even when the user scrolls down; and Sticky, which is similar to fixed, but sticks an element to a part of the page as you scroll down. There are also a few different displays. Inline, which allows two elements to be displayed next to each other and completely surrounds the element itself and doesn't stretch outside the element and cannot have their height and width altered; Block, which aren't displayed next to any other element and fill the entire width of the page by default, however you can alter height and width.; and Inline-Block, which allows elements to be displayed side by side and the height and width can be altered.

Property Name Description
position Position Changes the position of where an element sits on a page.
z-index Z-Index Specifies how far forward or how far back an element should appear on a page.
display Display Determines whether or not an element can share space with another element.
float Float Moves an element as far left or right as possible on a page.
clear Clear Specifies how elements should behave when they bump into each other on the page.

CSS Typography

Typography in CSS is the way the text is positioned and displayed. There are a few default fonts that come on most computers including Times New Roman, Courier, Impact, Arial, Georgia, etc. However, these options can be limiting. To add a font of your choosing to your website, either link it with a < link > tag in the HTML sheet or link it with the @font-face property in the CSS sheet.

Property Name Description
font-family Font Family Changes what font a text element has. Also adds fallback fonts.
font-weight Font Weight Changes how thick or thin the text element appears.
font-style Font Style Changes whether the text is bold, italic, or normal.
text-transform Text Transformation Transforms the text to be displayed as all UPPERCASE, all lowercase, or Normal
letter-spacing Letter Spacing Changes the amount of space between letters in a text element.
word-spacing Word Spacing Changes the amount of space between words in a text element.
line-height Line Height Changes the amount of space between each line containing text.
text-align Text Alignment Changes how the text is aligned on screen to the left, right, center, or justify.

CSS Colors

Colors in CSS can be displayed in a few different ways. There's named colors, default colors that have been named in CSS; RGB, displayed as rgb(0, 0, 0), which uses decimal numbers between 0 and 255 to represent the amount of red, green, and blue a color has; HSL, displayed as hsl(0, 0, 0), which represents the hue, saturation, and lightness of a color and has the first number be a number between 0 and 360, while the other two numbers are percentages; and Hexidecimal, which is where the amount of red, green, and blue in a color are represented by two letters/numbers each, in the form of #AA0011. Adding an "a" onto the end of RGB or HSL will add a value called "alpha", which allows you to alter the opacity in the color itself.

Property Name Description
color Color Changes the foreground color of an element.
background-color Background Color Changes the background color of an element.
opacity Opacity Changes how see-through an element is.
